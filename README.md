# Kalyan Mutyala Portfolio

A professional portfolio website showcasing expertise in **Generative AI**, **Artificial Intelligence**, and **Data Science**.

## 🎯 About

This is a personal portfolio website built to highlight skills and projects in:
- **Generative AI & Agentic AI**
- **Large Language Models (LLM) & RAG**
- **Python & AIML**
- **SQL & Data Analysis**
- **Tableau & Data Visualization**

The site features project showcases, technical skills, and links to professional profiles.

## 🛠 Tech Stack

### Frontend
- **HTML5** - Semantic markup
- **CSS3** - Responsive styling with SASS preprocessing
- **JavaScript** - Interactive features and smooth scrolling
- **jQuery** - DOM manipulation and effects
- **Scrollex** - Scroll-based animations

### Design
- **Font Awesome** - Icon library
- **Responsive Design** - Mobile-first approach with breakpoints
- **SASS/SCSS** - CSS preprocessing for maintainable styles

### Template
- Based on **Solid State** by HTML5 UP
- Licensed under Creative Commons Attribution 3.0

## 🚀 Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Scrolling** - Elegant scroll-based animations using Scrollex
- **Professional Layout** - Multi-section design showcasing projects and skills
- **Social Links** - Direct links to LinkedIn and GitHub profiles
- **Accessibility** - Semantic HTML and keyboard navigation support
- **Fast Loading** - Minified CSS and JavaScript assets

## 📱 Responsive Breakpoints

The site uses responsive design with breakpoints for:
- Desktop displays
- Tablets and small laptops
- Mobile devices

SASS breakpoint utilities ensure consistent responsive behavior across all sections.

## 🎨 Styling

All styles are built using SASS/SCSS for maintainability:
- **Base styles** - Reset, typography, and page defaults
- **Components** - Buttons, forms, cards, icons, and more
- **Layout** - Header, footer, banner, menu, and wrapper sections
- **Utilities** - Mixins, functions, and vendor prefixes

## 📝 Notes

This portfolio is designed to be easily customizable. To add new projects:
1. Add project sections to `index.html`
2. Style using SASS components in `assets/sass/`
3. Add project images to the `images/` folder
4. Compile SASS to CSS for deployment
